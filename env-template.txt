#    → Copy Client ID and Secret
#

#    → Run in terminal: openssl rand -base64 32
#    → Or use: https://generate-secret.vercel.app/32
#    → Copy and paste in NEXTAUTH_SECRET above

# 5. After configuring, run:
#    → pnpm db:push
#    → pnpm dev
